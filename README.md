# AI-Based-Resume-Screening-Job-Matching-System

An intelligent recruitment platform that automates resume screening, skill extraction, candidate classification, clustering, and job matching using Artificial Intelligence and Machine Learning techniques.

Features:
• Resume preprocessing and text cleaning using NLP
• Skill extraction using spaCy
• TF-IDF vectorization and dimensionality reduction (SVD)
• Candidate clustering using K-Means
• Candidate-job matching using BFS and A* Search
• Candidate classification and scoring using ANN (MLPClassifier)
• Match score prediction using Linear Regression
• Candidate ranking and visualization dashboards

Tech Stack:
Python, Pandas, NumPy, spaCy, Scikit-learn, Matplotlib, NLP, Machine Learning

Dataset:
2458+ resumes processed for candidate profiling, classification, clustering, and intelligent job-role matching.
